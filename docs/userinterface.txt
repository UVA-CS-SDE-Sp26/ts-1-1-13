PROGRAM DESCRIPTION

This program allows the user to run the TopSecret deciphering system from the command line.
The user may list available secret files, select a file to decipher using the default key,
or select a file and provide an alternate key file.

The TopSecret class serves as the command-line interface and delegates file handling
and decryption to helper classes (FileHandler and Cipher).
The program validates user input and prints clear error messages when necessary.


PSEUDOCODE

Program begins in main(String[] args)

If args.length == 0:
    Call FileHandler to list available secret files
    Print file numbers and filenames
    Print usage instructions

If args.length == 1:
    Attempt to parse args[0] as integer fileNumber
    If parsing fails:
        Print "Error: fileNumber must be an integer."
        End program

    Call FileHandler to retrieve file contents
    If fileNumber is invalid:
        Print error message
        End program

    Use Cipher with default key
    Print decrypted message

If args.length == 2:
    Attempt to parse args[0] as integer fileNumber
    keyFile = args[1]

    If parsing fails:
        Print "Error: fileNumber must be an integer."
        End program

    If keyFile does not exist:
        Print "Error: key file not found."
        End program

    Call FileHandler to retrieve file contents
    Use Cipher with provided keyFile
    Print decrypted message

If args.length > 2:
    Print usage instructions
    End program


COMMAND LINE USAGE

To list available files:
    java topsecret.TopSecret

To decipher a file using the default key:
    java topsecret.TopSecret <fileNumber>

To decipher a file using a custom key file:
    java topsecret.TopSecret <fileNumber> <keyFile>
